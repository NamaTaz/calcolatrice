PER IMPORTARE LA CALCOLTATRICE DEVI AVERE NETBEANS(Apache NetBeans IDE 16) E JAVA(vers. 1.8.0_192)!!!

- Entra in NETBEANS
- Clicca in alto a sinistra "File"
- Vai su "Import Project"
- Clicca "From ZIP..."
- Clicca "Browse.." che troverai sulla stessa linea di "ZIP File:"
- Cerca il file (dove lo hai salvato tu)
- Clicca sul file zip
- Clicca "Apri"
- Infine clissa "Import"
